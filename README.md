# INF232_TP_GROUPE18 - Santé Universitaire

Ce dépôt contient le socle du projet de TP INF232, incluant le générateur de données et la structure de dossiers.

## Structure du projet
- `main.py` : Script contenant le générateur de données (Tâche de Chanel).
- `data/` : Dossier destiné à recevoir le fichier CSV généré.
- `rapport/` : Dossier destiné à recevoir les graphiques et le rapport final.
- `README.md` : Mode d'emploi et répartition des tâches.

## Mode d'emploi
1. Installer les dépendances : `pip install pandas numpy`
2. Exécuter le générateur : `python main.py`
3. Le fichier `data/donnees_sante.csv` sera créé pour les analyses suivantes.
